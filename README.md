## Tech Stack

A financial website for transferring account balance to other users in the system. Using knowledge of authentication, relational database CRUD operations and transaction.

**Front-End**
- Vite
- React

**Back-End**
- Node.js
- Express

**Database**
- MySQL

**Features**
- Authentication System
- Account Balance
- Money Transferring
- Transaction History

**Routes**
- Homepage
- Login
- Register
- Transfer
- History
